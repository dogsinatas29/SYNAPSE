# 🎯 SYNAPSE v0.2.15 핵심 구현 목표 (Flowchart Enhancements)

## 1. Semantic Zooming (시맨틱 줌 - 시각적 깊이 제어)
- **줌 아웃 (Satellite View)**: 복잡한 내부 로직은 숨기고 큼직한 '수퍼 노드(클러스터)' 단위만 보여주어 전체 시스템의 뼈대를 파악하게 합니다.
- **줌 인 (Detail View)**: 다이아몬드(조건문), 헥사곤 노드 등 세부적인 로직과 흐름이 드러나도록 추상화 레이어를 구현합니다.

## 2. Milestone Anchor (마일스톤 앵커 - 핵심 흐름 수직 정렬)
- Auth(인증), DB_Process(DB 처리), Response(응답) 등 시스템의 **'핵심 체크포인트'**를 화면의 수직 축(Vertical Axis)에 기둥처럼 배치합니다.
- 상대적으로 덜 중요한 예외 처리나 부가 로직은 접어두거나(collapsible) 곡선으로 우회시켜, 눈이 메인 로직을 따라가기 편하게 레이아웃을 개선합니다.

## 3. Path Highlighting (경로 하이라이팅 - 활성 흐름 추적)
- 기본적으로 캔버스의 모든 노드와 선을 반투명하게(Dimmed) 처리합니다.
- 특정 노드에 **마우스를 올리거나(Hover) 로직 테스트 모드**가 될 때, 실제로 데이터가 흘러가는 '활성화된 경로(Active Path)'만 밝게 빛나며 맥박(Pulse)이 뛰는 듯한 애니메이션 효과를 부여합니다.

## 4. Graph 및 FLOW 엣지 하이라팅

- 마우스 호버 (Hover) - "빠른 탐색":
-- 마우스를 올린 노드와 연결된 경로(Edge)가 즉시 펄스 효과와 함께 밝아집니다 (Path Highlighting).
-- 현재 마우스가 위치한 곳의 연관 관계를 실시간으로 빠르게 훑어볼 때 사용합니다.
-- 이 기능은 Graph 모드와 Flow 모두 둘다 지원합니다

- 노드 선택 (Select/Click) - "집중 모드 (Focus Mode)":
-- 특정 노드를 클릭하여 선택하면, 해당 노드와 직접적인 의존성(1단계)을 가진 노드들만 밝게 남고 나머지는 반투명하게 처리됩니다.
-- 선택된 상태에서는 마우스를 떼어도 하이라이트가 유지되므로, 특정 로직을 고정해서 분석하거나 [V] 승인 / [X] 거절과 같은 제어 명령을 내릴 때 사용합니다.
-- 이 기능은 Graph 모드에서만 지원합니다. 순서도에서는 각 항목의 선택이 되지 않으므로 이 기능은 필요가 없습니다. 

 결론적으로, **전체적인 흐름을 빠르게 확인하려면 마우스를 움직여(Hover) 보시고, 특정 로직에 집중하여 작업을 수행하려면 노드를 선택(Select)**하시면 됩니다.

## 📝 Post-Work Log (작업 후 기록)

### ✅ 완료된 시각적 아키텍처 개선
- **Semantic Zooming 엔진 구현**: 줌 레벨(LOD)에 따라 노드의 상세 정보(라벨, 아이콘, 내부 요약) 노출도를 동적으로 조절합니다.
- **Path Highlighting (Active Path)**: 엣지와 노드에 펄스 애니메이션을 적용하여 현재 호버/선택된 로직의 흐름을 시각적으로 강조합니다.
- **Flow View 비선형 로직 분석**: 단순 수직 나열이 아닌 `if/loop` 등을 인식한 실제 제어 흐름 순서도를 완성했습니다.

### 🛠️ 기술적 최적화 및 버그 수정
- **Coordinate Drift 해결**: `renderNode` 내 `save/restore` 불균형 및 중복 좌표 변환 버그를 수정하여 노드 가시성 문제를 원천 해결했습니다.
- **CPU Performance 최적화**: 무거운 그림자 효과를 활성 경로에만 제한하고, 불필요한 전체 redraw를 제거하여 저사양 사령관님들의 환경에서도 쾌적하게 작동합니다.
- **Node Diet 원칙 강화**: 접힌 클러스터 내의 노드 렌더링을 완전히 스킵하여 연산 효율을 높였습니다.
