# Release Notes v0.1.7 - SYNAPSE Visual Architecture

## [English] Core Delivery: Multi-MD Strategy & Intelligent Logging

This version marks the stabilization of SYNAPSE's most critical features, ensuring they work seamlessly in your production environment.

### ğŸš€ Key Features

1. **Multi-MD Strategy Support**
   - SYNAPSE now intelligently scans your workspace for architectural context.
   - Automatically identifies and links `prompts/*.md` (history) and other `.md` files as live documentation.
   - Visualizes design decisions alongside source code, providing a comprehensive view of "Why" vs "How".

2. **Zero-Friction Prompt Logging (`Ctrl+Alt+M`)**
   - Capture design thoughts instantly without leaving your flow.
   - **Global Activation**: Use the shortcut even on the Welcome page or empty editors.
   - **Auto-Git Integration**: Logged prompts are automatically saved as Markdown in `prompts/` and staged for Git.
   - **Real-time Canvas Sync**: The architecture canvas refreshes instantly when a new prompt is logged.

### ğŸ”§ Stability & Deployment Improvements
- **Self-Contained Package**: Fixed `.vscodeignore` to include all runtime dependencies (`node_modules`) inside the VSIX.
- **Reliable Activation**: Enhanced `activationEvents` to ensure SYNAPSE commands are always ready to fire.
- **Debug Traceability**: Added `[SYNAPSE]` console logs for easier troubleshooting.
