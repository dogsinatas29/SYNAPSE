## [ENG]
**SYNAPSE v0.2.17 is a major update consolidating system stability, canvas UX, and AI orchestration features.**

### 1. AI Orchestration & DTR Control
- **DTR Visualization:** Instantly projects AI reasoning depth via purple glow effects on high-density nodes.
- **Inference Pressure Gauge:** Monitor and control AI DTR depth from the status bar. Edge thickness and pulse animations respond dynamically to DTR levels.
- **AiOrchestrator Integration:** Dynamically adjusts LLM parameters (Temperature, Top_P) based on DTR values to ensure optimal coding reasoning.

### 2. WYSIWYG Logic Editing
- **Drawing equals Coding:** Adding/deleting nodes and connecting edges on the canvas directly manipulates the workspace files and internal logic.
- **Auto-Injected Imports:** Confirming an edge (via the `?` badge) triggers native `import` or `require()` injection at the top of the source file.
- **Safe File Deletion:** Replaces destructive deletion with a `// [SYNAPSE_DELETED]` wrapper and automated snapshots, supporting secure file resurrection.

### 3. Stability & Data Hygiene
- **Anti-Resurrection Logic:** Prevents deleted nodes from reappearing due to stale summaries or cache. Uses `SequentialTaskQueue` and forced state snapshots to seal deletions.
- **Unique ID Integrity:** Implements salted hashing (Source+Target+Time) for edge IDs to prevent collisions and vanishing artifacts during high-speed editing.
- **Improved Parsing:** Enhanced `GEMINI.md` regex to correctly identify nodes even when styled with markdown bold or backticks.

### 4. UX & Performance
- **Expanded Hit Padding:** Added `12px` hit-padding to all nodes and loosened edge-click requirements for smoother interaction.
- **Smart Layouts:** Introduced auto-correction for node overlaps and set the `Documentation Shelf` to be collapsed by default to save space.
- **History Scrollbar:** Added a dedicated vertical scrollbar to the snapshot history panel for easier navigation.
