## [ENG]
**SYNAPSE v0.2.17 focuses on core stability, canvas UX, and structural design convenience.**

### Key Changes
- **DTR Engine Integration:** Visualize AI reasoning depth through purple glow effects on high-density nodes.
- **Inference Pressure Gauge:** Monitor and control AI DTR depth directly from the VS Code status bar.
- **Enhanced Canvas Navigation & Stability:** Added scrollbars and resolved multiple critical load-time errors (`TypeError`, `RangeError`).
- **Data Hygiene Improvements:** Upgraded `project_state.json` file parsing and introduced `decycle` plain-data mechanism for serialization.
- **Advanced Logic Edit Mode:** Creating new nodes and connecting files via GUI now seamlessly integrates with physical workspace files (creation & deletion).
- **Snapshot & Rollback System:** Secure backend file manipulation with automatic snapshot backups before deletion, supporting file resurrection.
- **Safe Node Deletion:** Rather than destructive file deletion, source code files are now safely disabled using a `// [SYNAPSE_DELETED]` comment wrapper.

### Visualization & UX Improvements
- **Default Collapsed UI:** Modules like `Documentation Shelf` and `Intelligent Context Vault` now render as collapsed by default for enhanced readability.
- **Node Overlap Prevention:** Implemented an auto-correction engine that ensures minimum distance (150px/100px) between auto-discovered nodes upon canvas load.
- **Expanded Hit-Boxes:** Loosened edge-click requirements and added a `12px` hit-padding to all nodes, making drag-and-drop operations much smoother.
- **Deprecated Rigid Guides:** Completely removed the redundant `[V]` / `[X]` (Approve/Reject) buttons above proposed nodes in favor of the new Edge UI, returning to a cleaner visual aesthetic.

### Edge Connectivity & Code Injection
- **Intelligent Garbage Selector:** Introduced an inline `âŒ (Trash)` button directly on edges for easy and precise removal of accidental connections.
- **`?`/`!` Edge Confirmation Badges:** Pending edges from Logic Edit mode now display a yellow `?` badge for user review.
- **[New] Auto-Injected Imports on Confirmation:** Clicking the `?` badge to confirm an edge architecture now intercepts the workspace. SYNAPSE will auto-detect the language (`py`, `ts`, `js`) and natively inject the corresponding `import` or `require()` statement directly at the very top of the calling source file (`fromFile`). The ultimate manifestation of the WYSIWYG coding philosophy: drawing the logic connects the code.
