## [ENG]
**SYNAPSE v0.2.18 is an immediate hotfix release correcting UI/Backend mismatch issues regarding node creation persistence and import injection failures found in v0.2.17.**

### Key Changes
- **Completely resolved "Ghost Node/Edge" occurrences:** 
  - Fixed a critical logical error where newly dropped nodes from the '+ Node' UI failed to bypass backend sanity checks during position saving, wiping them completely on canvas refresh. UI nodes now force append themselves safely to the backend configuration.
- **Robust Auto-Imports for Novel Connections:** 
  - Hotfixed a situation where newly created edges initially lacked internal workspace file paths (`_fromFile`). VS Code now utilizes a dynamic backend fallback to pinpoint and inject these missing file parameters before committing the edge connection. 
- Disappearing edge artifacts have effectively been mitigated through synchronized JSON rendering.
