# SYNAPSE Release Notes - v0.2.20
[Status: Verified by Commander]

## 🚀 주요 변경 사항

### 1. DTR (Deep-Thinking Ratio) AI 오케스트레이션 통합
- **Inference Pressure Control**: DTR 값(0.0 ~ 1.0)에 따라 LLM의 `temperature`와 `top_p`를 동적으로 조절합니다.
  - **High DTR (0.7+)**: 시니어 엔지니어 모드. 적대적 시나리오 검토 지연 프롬프트 주입 및 낮은 Temperature(0.15) 적용.
  - **Low DTR (0.3)**: 빠른 프로토타이핑 모드. 높은 Temperature(0.7) 적용.
- **Visual Feedback**: 캔버스의 모든 엣지가 DTR 값에 따라 두께와 발광 강도가 실시간으로 변화하여 시스템의 '논리적 압력'을 시각적으로 표현합니다.

### 2. 엣지 관리 및 Python 지원 강화 (Bug Fixes)
- **Edge Deletion Bypass**: 소스 코드 내 import 구문을 찾지 못하더라도 캔버스 및 `project_state.json`에서 연결을 강제로 끊을 수 있도록 수정하여 엣지가 지워지지 않던 버그를 해결했습니다.
- **Virtual Node Import**: 확장자가 없는 가상 노드(예: `TEST_NODE_DB`)를 Python 파일이 참조할 때 올바른 `import` 문이 생성되도록 지원합니다.
- **Empty File Injection**: 빈 파일에 첫 엣지를 연결할 때 최상단에 import 문이 안전하게 삽입되도록 로직을 보강했습니다.

## 🛠️ 기술 세부 사항
- `src/core/AiOrchestrator.ts`: 인퍼런스 파라미터 및 프롬프트 접두사 관리 코어.
- `src/webview/CanvasPanel.ts`: DTR 정보 동기화 및 엣지 삭제 예외 처리.
- `ui/canvas-engine.js`: DTR 비례 엣지 렌더링 ($\rho$ 가중치 적용).
