# SYNAPSE v0.2.12 설치 가이드

## 📦 VSIX 설치 방법

### 방법 1: 터미널 (권장)
```bash
code --install-extension synapse-visual-architecture-0.2.12.vsix
```

### 방법 2: VS Code UI
1. 왼쪽 **Extensions** 패널 열기
2. 우측 상단 `…` → **Install from VSIX…**
3. `synapse-visual-architecture-0.2.12.vsix` 선택
4. 설치 완료 후 **Reload Window**
