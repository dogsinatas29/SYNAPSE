# SYNAPSE Release Notes - v0.1.8

## 🚀 What's New
이번 버전에서는 `GEMINI.md`와 실제 코드 사이의 연결성을 극대화하는 LSP(Language Server Protocol) 기능이 대폭 강화되었습니다.

### 1. LSP 기능 강화 (Editor Integration)
*   **Hover 정보 표시**: `GEMINI.md` 내의 파일 경로(`📄 path/to/file`) 위에 마우스를 올리면 파일 참조 정보와 안내 툴팁이 표시됩니다.
*   **Go to Definition**: `Ctrl + Click`을 통해 설계 문서에서 실제 소스 코드로 즉각 이동할 수 있습니다.
*   **실시간 유효성 검사 (Diagnostics)**: `GEMINI.md`에 적힌 파일이 실제로 존재하지 않을 경우 에디터에서 경고(Warning)를 표시하여 설계와 실제 구현의 불일치를 방지합니다.

### 2. 안성성 및 빌드 개선
*   **Build Optimization**: `tsconfig.json` 설정을 최적화하여 빌드 시 불필요한 아티팩트가 포함되는 문제를 해결하고 컴파일 속도를 개선했습니다.
*   **Version Sync**: 확장 프로그램 활성화 및 상태 표시줄의 버전 정보가 0.1.8로 동기화되었습니다.

## 🛠 설치 및 업데이트 방법
1. 기존 VSIX를 제거하거나 VS Code를 재시작합니다.
2. 새 버전의 VSIX를 설치하거나 `npm run compile`을 실행하여 최신 변경 사항을 적용합니다.
3. `GEMINI.md` 파일을 열어 강화된 편집 기능을 경험해 보세요!
