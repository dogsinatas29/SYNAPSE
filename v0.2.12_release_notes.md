## 🆕 새로운 기능

### 1. 🧠 Context Vault (CTRL+ALT+M)
SYNAPSE의 컨텍스트 캡처 기능이 완전히 새로워졌습니다.

**저장 내용 (Before → After)**
| Before | After |
|--------|-------|
| 태그 선택 3단계 UI | **2단계** (입력 → 저장방식) |
| ADR 구조화 포맷 | **내 명령 + Git Diff 요약** |
| 복잡한 메타데이터 | **실용적인 변경 이력** |

**저장 포맷 예시 (`prompts/context.md`):**
```markdown
## 📅 2026-02-22 오후 1:04

### 💬 명령
FlowScanner의 Python 인덴트 파싱 로직 수정해줘

### 📝 변경 요약
```diff
 src/core/FlowScanner.ts | 12 +++---
 src/extension.ts        |  5 +--
 2 files changed, 17 insertions(+), 5 deletions(-)
```
```

### 2. 🗂️ Context Vault 클러스터 (그래프 뷰)
캔버스 우측에 **🧠 Context Vault** 클러스터가 자동으로 표시됩니다.

- `prompts/` 디렉터리의 `.md` 파일 목록을 노드로 시각화
- **Read-only**: 삭제 불가, 숨김(접기)만 가능
- `project_state.json`에 저장되지 않는 **휘발성 클러스터** (매번 실시간 생성)
- 노드 클릭으로 해당 컨텍스트 파일 열기

### 3. 📋 Multi-MD Bootstrap 선택기 (CTRL+ALT+P)
`Bootstrap from GEMINI.md` → **`Bootstrap from MD…`** 로 업그레이드.

`CTRL+ALT+P` 또는 명령 팔레트에서 실행하면:
```
┌─────────────────────────────────────────────────────────┐
│ $(file) GEMINI.md      기본 아키텍처 설계 문서           │
│ $(file) CLAUDE.md                                       │
│ $(file) README.md                                       │
│ $(zap) Lite Bootstrap  MD 없이 프로젝트 구조 자동 탐색   │
└─────────────────────────────────────────────────────────┘
```
- 워크스페이스 루트의 모든 `.md` 파일 자동 탐색
- `GEMINI.md` 있으면 최우선 상단 표시
- LLM 종류에 무관하게 어떤 MD 파일로도 부트스트랩 가능
