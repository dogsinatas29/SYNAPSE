## 🆕 새로운 기능

### 1. 🧠 Intelligent Context Vault 

SYNAPSE의 컨텍스트 캡처 기능이 '토글 레코딩(Toggle Recording)' 방식으로 완전히 개편되었습니다.

**레코딩 워크플로우:**
1. **[CTRL+ALT+M] 또는 캔버스 [⏺ REC] 클릭:**
   - 즉시 `.synapse_contexts/` 폴더에 `YYYY-MM-DD_HHMM.md` 세션 파일 생성
   - 상태바 우측에 🔴 **REC** 인디케이터 표시
   - 캔버스 툴바에 빨간색 **⏹ STOP** 애니메이션 버튼 표시
   - 캔버스 우측 '🧠 Intelligent Context Vault' 클러스터에 새 노드 실시간 반영

2. **작업 수행:**
   - LLM에 명령(Prompt)을 내리고 코드 수정
   - 필요한 텍스트를 에디터에서 선택(Select) 하거나 클립보드에 복사

3. **[CTRL+ALT+M] 또는 [⏹ STOP] 클릭:**
   - 레코딩 중지 및 UI 원상 복구
   - 에디터 선택 텍스트(또는 클립보드 내용)를 사용자 명령으로 기록
   - 변경된 파일 내용(`git diff --stat`)을 캡처하여 세션 파일에 자동 추가
   - 화면 우하단 "✅ Context 저장 완료" 팝업 표시

### 2. 🗂️ Intelligent Context Vault 클러스터 (그래프 뷰)
캔버스 우측에 **🧠 Intelligent Context Vault** 클러스터가 자동으로 표시됩니다.

- `.synapse_contexts/` 디렉터리의 `.md` 파일 목록을 노드로 시각화
- **Read-only**: 삭제 불가, 숨김(접기)만 가능
- `project_state.json`에 저장되지 않는 **휘발성 클러스터**
- 노드 클릭으로 해당 컨텍스트 파일 열기

### 3. 📋 Multi-MD Bootstrap 선택기 (CTRL+ALT+P)
`Bootstrap from GEMINI.md` → **`Bootstrap from MD…`** 로 업그레이드.

- 워크스페이스 루트의 모든 `.md` 파일 자동 탐색
- `GEMINI.md` 있으면 최우선 상단 표시
- LLM 종류에 무관하게 어떤 MD 파일로도 부트스트랩 가능
