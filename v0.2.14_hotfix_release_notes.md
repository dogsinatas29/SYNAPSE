# SYNAPSE Release Notes - v0.2.14 Hotfix

## [v0.2.14 Hotfix] - 2026-02-28
### War Room: Globe Layout Destruction & Viewport Memory Sync
**"모든 엣지는 코드의 진실을 가리키며, 시선은 결코 흔들리지 않아야 한다."**

이번 v0.2.14 Hotfix(긴급 패치)는 Flow/Graph View의 치명적인 문제였던 "지구본 현상(가로 퍼짐)"과 렌더링 시 "시선 튕김(Viewport Bouncing)" 문제를 완전하게 해결했습니다.

### 🚀 주요 변경 사항

#### 1. 지구본 파괴 작전: Rank-based Node Positioning
- **Topological Rank (위상 정렬 적용)**: 모든 노드에 In-degree(진입 차수) 기반의 서열(Rank)을 부여하여 상하 선후 관계를 명확히 했습니다. 
- **수직적 폭포수 레이아웃**: 기존의 단순 Grid 좌표 계산과 무분별한 가로 팽창을 제거하고, 연관성이 있는 노드들만 가로로 묶인 뒤 `Level`이 떨어질수록 아래로 흘러내리는 수직 폭포수 형태(DAG)의 진정한 트리형 그래프 구조로 재설계되었습니다.

#### 2. 컨텍스트 유지: Viewport Memory System
- **시선 고정(State Persistence)**: 캔버스의 데이터가 갱신(`projectState` 이벤트)될 때마다 무조건적으로 `fitView()`를 때려버리던 악습을 제거했습니다.
- **연속적 조망 보장**: 이미 화면에 노드가 떠있는 상태에서 추가/수정/테스트 로직이 돌 경우, 현재 사용자가 보고 있는 좌표(X,Y)와 줌(Zoom) 배율을 강제로 기억시켜 시야의 연속성을 보장합니다.
