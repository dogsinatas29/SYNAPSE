# SYNAPSE Release Notes - v0.2.15

## [v0.2.15] - 2026-02-22
### War Room: Legacy Logic Fix & Flow Precision
**"모든 엣지는 코드의 진실을 가리켜야 한다."**

이번 v0.2.15 패치는 Flow View의 치명적인 문제였던 "단순 선형 시각화"를 해결하고, 코드의 실제 논리 구조를 완벽하게 반영하도록 엔진을 리팩토링했습니다.

### 🚀 주요 변경 사항

#### 1. Flow Scanner 리팩토링 (Flow Logic Fix)
- **비선형 흐름 지원**: 이제 `if/else`, `while`, `for`, `try/except` 등의 제어문을 인식하여 갈래가 나뉘는 순서도를 생성합니다.
- **Stack-based Analysis**: 구문 수준의 스캐닝을 통해 블록의 시작과 끝을 정확히 파악하여 엣지를 연결합니다.
- **Join Point 시스템**: 분기된 로직이 끝나는 지점에 보이지 않는 병합 포인트를 생성하여 흐름의 정합성을 유지합니다.

#### 2. Canvas Engine 업데이트 (Visual Support)
- **Hidden Node 처리**: 흐름 보조를 위한 숨겨진 노드들을 렌더링에서 제외하되 연결 성능은 유지하도록 최적화했습니다.
- **다중 언어 스캐너 강화**: Python뿐만 아니라 JS, C++, Rust 스캐너에도 비선형 로직 분석을 일괄 적용했습니다.

#### 3. UX 개선
- **Bypass 로직 시각화**: `main.py`의 DB 조회 후 Bypass 여부에 따른 분기가 이제 정확하게 시각화됩니다.
